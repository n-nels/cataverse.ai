# `src/analysis/` Overview

High-level guide to the analysis modules. This file is intentionally brief and
focuses on module responsibilities rather than algorithm details.

## Module Responsibilities

- `main.py`: Orchestrates the analysis pipeline for a spectrum file, coordinating
  spectral fitting, kinetics, and peak height extraction in the legacy order.
- `spectral_fitting.py`: Performs Voigt-profile spectral fitting and produces
  peak parameters used for downstream processing.
- `kinetics_fitting.py`: Fits kinetic models using cumulative peak area data.
- `peak_heights.py`: Computes peak heights and summary metrics for reporting.
- `output.py`: Writes analysis outputs (CSV artifacts, plots where applicable)
  with legacy filenames and schemas.
- `io.py`: Handles loading spectrum data, validating input structure, and
  normalizing data representations before fitting.
- `integrate_ir_iso_xchg.py`: Specialized isotopic exchange processing utilities
  retained for compatibility with legacy workflows.

## Notes

- Configuration is driven by `config/analysis.yaml` via `src.core.config`.
- Legacy output filenames and schemas are preserved.
